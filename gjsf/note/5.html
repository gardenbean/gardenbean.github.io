# 五 外国人的民事法律地位

外国人民事法律地位指外国自然人或法人在内国享有民事权利和承担民事义务的法律状况。

赋予内外国人平等的法律地位，是国际私法得以产生的一个重要前提。

待遇制度：

  1. 国民待遇：内外国人同等的民事权利地位
  2. 最惠国待遇：将已经给予或将来给予第三国的公民或法人的优惠同样给予缔约他方
  3. 歧视与非歧视待遇：把不给予本国或其他外国的限制性规定专门适用于特定国家（或不给优惠）
  4. 互惠待遇：赋予外国人某种优惠时，要求它的公民在外国人所属国享受同样的优惠
  5. 普遍优惠待遇：发达国家给发展中国家出口制产品一种普遍的、非歧视的和非互惠的关税优惠制度

最惠国待遇例外：

  1. 给予邻国的特权和优惠
  2. 边境贸易和运输方面的
  3. 有特殊历史关系形成的特定地区的
  4. 经济集团内部的（如欧盟）

外国人在中国的民事法律地位的演变

  1. 合理待遇时期
  2. 闭关锁国时期
  3. 特权时期
  4. 平等待遇时期

外国人在中国享有多方面的人身权和财产权：

  1. 人身保护权
  2. 亲属权（中外或外外婚姻，收养）
  3. 继承权
  4. 劳动权
  5. 智力成果权
  6. 经营工商
  7. 可取得土地的长期租赁使用权
  8. 司法保护权
