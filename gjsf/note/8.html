# 八 法律行为与代理

法律行为：指当事人通过意思表示，设定变更消灭特定权利义务关系的重要法律事实。

代理：指代理人在权限内，以本人名义实施民事法律行为，对本人发生效力。

代理权关系准据法适用范围：

  1. 代理人的权限
  2. 报酬数额
  3. 终止代理关系条件
  4. 代理关系是否因本人死亡或受禁治产宣告而消灭
  5. 狭义无权代理人应负的责任

委托代理的法律适用多采用意思自治原则，未选择时的主张：

  1. 适用代理关系成立地法
  2. 适用代理行为地法
  3. 适用代理人营业地法或住所地法
  4. 适用代理合同的重心地法或最密切联系法

我国规定代理适用代理行为地法，但本人与代理人之间适用代理关系发生地法。
