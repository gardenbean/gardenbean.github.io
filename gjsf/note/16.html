# 十六 国际民事诉讼

国际民诉法渊源：

  1. 国内立法
  2. 国际条约

诉讼费用担保：指外国人或在内国未设有住所的人在内国法院提起民诉时，应被告请求或依内国法规定，为防止其滥用诉讼权利，或防止败诉后不支付诉讼费用，而由内国法院责令原告提供的担保。

国际民事管辖权：指一国法院对特定的涉外民事案件行使审判权的资格。
分为：

  1. 直接管辖权：
  2. 间接管辖权：

确定国际民事管辖权原则：

  1. 地域管辖为主，属人管辖为辅
  2. 冲突协调和有效行使
  3. 法定管辖与当事人自主选择管辖相结合
  4. 互惠原则
  5. 便利诉讼和防止滥用诉讼权利

直接管辖权可分为：

  1. 普通管辖和特别管辖（以被告住所为管辖依据）
  2. 专属、平行和排除（以法律直接规定和任意选择为标准）
  3. 法定、协议和应诉（以是否允许选择法律规定以外的法院为标准）

专属管辖：指对某些有特别性质的涉外民事案件强制规定只能由特定国家的内国法院行使独占排他的管辖。（国家安全法，本国不动产，本国专利）
平行管辖：指国家对某些案件具有管辖权的同时，并不否认外国法院对此类的管辖权。（可选择内国或外国起诉，合同或财产纠纷）
排除管辖：与专属相对立，跟内国领土公民无关。

应诉管辖：指原告向法定管辖以外本无管辖权的法院起诉，但被告应诉未对管辖表示异议所形成的管辖权。

合同无效、被撤销或终止的，不影响合同中独立存在的有关解决争议方法的条款的效力。

平行诉讼：指相同的当事人基于同一诉讼请求，向两个或两个以上的国家法院起诉，而两个或两个以上的国家法院都进行受理的情形。
产生原因：客观原因是各国法律关于国际民事诉讼管辖权规定不同，主观原因是当事人挑选法院
解决：不方便法院原则和颁发禁止诉讼令

我国国际民事管辖权规定

  • 普通管辖：只要被告住所地或经常居住地在中国境内
  • 特别管辖：（23-32）
    合同纠纷，由被告住所地、合同履行地法院
    保险合同，由被告住所地、保险标的物所在地法院
    票据纠纷，由被告住所地、票据支付地法院
    公司纠纷，由公司住所地法院（股东资格、分配利润、解散）
    运输合同，由被告住所地、运输始发地目的地法院
    侵权行为，由被告住所地、侵权行为地法院
    交通事故，由被告住所地、事故发生地、最先到达地法院（请求赔偿）
    海事事故，由被告住所地、碰撞发生地、最先到达地、加害船扣留地法院（请求赔偿）
    海难救助，由救助地、船最先到达地法院（费用）
    共同海损，由船最先到达地、共同海损理算地或航程终止地法院
    注：对不在中国居住或下落不明的人提起有关身份关系的诉讼，由原告住所地法院
    注：因合同纠纷或其他财产权益纠纷对在中国没有住所的被告提起诉讼，可由合同签订地、合同履行地、诉讼标的物所在地、可供扣押财产所在地、侵权行为地、代表机构住所地法院管辖
  • 专属管辖：
    中外合作经营合同
    中国境内不动产
    中国港口作业纠纷
    遗产纠纷，被继承人死亡时住所地或主要遗产所在地在中国境内
  • 协议管辖：（专管案件不得选择，仲裁除外）
    未提出管辖异议并应诉答辩的，视为受诉法院有管辖权（默示）
    海诉地管
    海事侵权，除 29-31 外，还可由船籍港海事法院
    海运合同，除 28 外，还可由转运港海事法院
    海船租用合同，由交船港、还船港、船籍港、被告住所地海事法院
    海上保赔合同，由保赔标的物所在地、事故发生地、被告住所地海事法院
    海船劳务合同，由原告住所地、合同签订地、船员登船港或离船港、被告住所地海事法院
    海事担保，由担保物所在地、被告住所地海事法院
    船舶抵押，还可由船籍港海事法院
    船舶所有权、占有权、使用权、优先权纠纷，由船舶所在地、船籍港、被告住所地海事法院
    认定海上财产无主，由财产所在地海事法院
    海诉专管
    港口作业，由港口所在地海事法院
    海域污染，由污染发生地、损害结果地、采取预防地海事法院
    海域勘探合同纠纷，由合同履行地海事法院
    海诉协管
    书面协议选择中国海事法院，即使与纠纷实际联系的地点不在中国领域内，也有管辖权

涉外民事案件的一审法院是基层法院，重大的才由中级法院
专利案件由知识产权法院、最高法院确定的中级法院和基层法院
海事案件由海事法院

有关平行诉讼规定

  1. 一方在国外起诉，一方在国内起诉，可予受理
  2. 判决后，外国法院请求承认，不予准许，但有条约规定的除外
  3. 外国法院判决已被承认的，不予受理

不方便法院原则：（涉外民事案件同时符合以下情形，可裁定驳回）

  1. 被告提出管辖异议
  2. 当事人之间不存在选择中国法院管辖协议
  3. 案件不属于中国法院专属管辖
  4. 案件不涉及中国国家、公民、法人利益
  5. 案件主要事实不是发生在中国境内，审理案件在认定事实和适用法律方面存在重大困难
  6. 外国法院对案件享有管辖权，且审理更加方便

国家及其财产享有豁免权：

  1. 司法管辖豁免
  2. 诉讼程序豁免
  3. 强制执行豁免

期间：由法律规定的或法院依职权指定的，法院、当事人或其它诉讼参加人为一定诉讼行为的时间期限。

国际货物买卖合同的时效期限为 4 年。

诉讼保全：指法院对判决可能难以执行，根据对方申请或依职权，可裁定对其财产证据保全，作出或禁止一定行为。

  1. 财产保全
  2. 证据保全
  3. 行为保全

域外送达：指一国法院将司法文书交给居于国外的诉讼当事人或其他诉讼参与人的行为。

域外取证方式：
直接取证：

  1. 领事取证（我国采取）
  2. 特派员取证
  3. 当事人或诉讼代理人自行取证

间接取证：请求书方式

承认和执行外国判决的一般条件：

  1. 必须是民事判决
  2. 原判决国法院必须有合法管辖权
  3. 已经生效或具有执行力
  4. 诉讼程序是公正的
  5. 判决必须是合法取得的
  6. 不存在诉讼竟合（判决相冲突）
  7. 不违背内国公共秩序

互惠形式：

  1. 法律互惠
  2. 事实互惠：有先例，且与该案件在性质和社会影响相似（我国采用）
  3. 推定互惠（只要不能证明有拒绝的承认和执行的事实，就推定互惠关系存在）

中国没有缔结或共同参加国际条约，也没有互惠关系的，裁定驳回申请，但申请承认外国法院的离婚判决除外。离婚判决中的夫妻财产分割、生活费负担、子女抚养方面不适用。
截至目前，韩德美新以等国均基于互惠原则承认或执行了我国法院的判决。
