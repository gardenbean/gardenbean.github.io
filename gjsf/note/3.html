# 三 冲突规范与法律选择

冲突规范：指某一涉外民事关系应适用哪一国法律的规范。

准据法特点：

  1. 按照冲突规范的指定所援用的法律
  2. 能确定当事人权利与义务的实体法

冲突规范类型：

  1. 单边，直接规定只适用内国法或只适用外国法
  2. 双边，规定连结点，取决连结点在内国还是外国（最基本）
  3. 重叠适用准据法的，同时适用两个或两个以上连结点所指各国家的法律
  4. 选择适用准据法的，只需选择适用其中一个连结点所指国家的法律
    • 无条件（主次或先后顺序之分）
    • 有条件

准据法表述公式：指在内外法的选择上，由各种具有双边意义的连结点来指引应适用的准据法的公式。

  1. 属人法（婚姻继承，经常居所地）法人为国籍国法
  2. 物之所在地法（不动产）
  3. 行为地法（合同缔结地，履行地，侵权地，婚姻缔结地，立遗嘱地）
  4. 法院地法（诉讼程序）
  5. 旗国法（船舶飞机）
  6. 当事人合意选择法（意思自治，确定涉外合同准据法的首要原则）
  7. 最密切联系法

连结点：指冲突规范法律问题应适用何种法律所依据的一种事实因素。
连结点法律意义：

  1. 从形式上看，是冲突规范中将范围中所指法律关系与某一法律联系起来的一种纽带
  2. 从实质上看，这种纽带又反映了该法律关系与某一法律之间存在着内在的、实质的或合理的联系

法律选择方法：

  1. 依法律性质
  2. 依法律关系性质
  3. 依最密切联系原则
  4. 依利益寻向
  5. 依案件应取得的结果
  6. 依有利于判决在外国得到承认与执行和有利于求得判决一致
  7. 依当事人自主意思

识别：指依据一定的法律观点，对有关事实的性质作出定性，归入特定的法律范畴，确定应援引哪一冲突规范的法律认识过程。
识别冲突：指依据不同国家的法律观点对有关事实定性所产生的抵触。

识别冲突产生原因：

  1. 对同一事实赋予不同的法律性质
  2. 对同一法律概念内涵理解不同
  3. 相同内容的法律问题分配到不同的法律部门
  4. 法律概念是另一个国家所没有的情况

识别依据：

  1. 法院地法说（我国采用）
  2. 准据法说
  3. 分析法学与比较法说
  4. 个案识别说
  5. 功能识别说
