# 十七 国际商事仲裁

国际商事仲裁特性：

  1. 仲裁机构有民间团体性质
  2. 管辖权建立在仲裁协议
  3. 仲裁程序一般不公开
  4. 当事人自治性较大
  5. 一裁终局制

仲裁法律渊源：《承认及执行外国仲裁裁决公约》（纽约公约）最有影响

仲裁方式分类：

  1. 临时与机构（以仲裁机构的组成形式）
  2. 依法与友好（以是否必须按照法律，善良公平原则）
  3. 私人间与非私人间（以双方是否为私人）私人间最为普遍

中国涉外仲裁机构：

  1. 中国国际经贸仲裁委员会
  2. 中国海事仲裁委员会
  3. 香港国际仲裁中心
  4. 中国新组建的仲裁委员会

仲裁协议的法律效力：

  • 已签字：
    1. 对双方具有严格约束力，不得再向法院提起诉讼
    2. 具有排除有关国家法院管辖权的效力
    3. 是有关仲裁机构行使仲裁管辖权的依据
    4. 是强制执行仲裁裁决的依据
  • 未签字：（仲裁条款仍具有法律约束力的情况）
    1. 法人的合并与分立
    2. 合同转让
    3. 提单转让和租约仲裁条款并入提单
    4. 国际公约中的仲裁条款的适用
    5. 代位清偿

最终裁决一经作出，一般既不能向法院起诉，也不能请求其他机构变更仲裁裁决。

我国仲裁法规定：
对仲裁机构：

  1. 没有约定仲裁机构或约定不明确，后又无法达成补充协议的，该仲裁协议无效。
  2. 机构名称不够准确，但能确定具体仲裁机构的，应当认定选定了仲裁机构。
  3. 仅约定仲裁规则，视为未约定仲裁机构，但有补充协议的或按仲裁规则能确定仲裁机构的除外。
  4. 仲裁协议有两个以上仲裁机构的，不能达成一致的，仲裁协议无效。
  5. 约定可仲裁也可起诉的，仲裁协议无效。但一方申请仲裁，另一方未在首次开庭前提出异议的除外。

对仲裁效力：（由中级法院或专门法院，向两个法院申请的，由最先立案的法院）

  1. 对仲裁效力有异议的，可请求仲裁委员会或法院裁定，并应当在仲裁庭首次开庭前提出。
  2. 如一方请求仲裁委员会，另一方请求法院，由法院裁定。仲裁机构作出决定后，法院不予受理。

仲裁协议独立存在，合同的变更解除终止，不影响仲裁协议的效力。
法院受理仲裁案件后，作出裁定前，申请人可撤回；作出裁定后即发生效力。

有以下情形裁定撤销：

  1. 合同中没有订有仲裁条款或事后没有达成书面仲裁协议
  2. 被申请人没有得到仲裁程序通知，或其他不属于被申请人原因未能陈述意见的
  3. 仲裁庭组成或程序规则不符的
  4. 裁决事项不属于仲裁协议范围或仲裁机构无权仲裁的（以超裁为由申请撤消的，法院应当撤消超裁部分；与其他事项不可分的，应当撤消裁决）

《纽约公约》裁决在中国的承认与执行

  1. 互惠保留和商事保留
  2. 由被执行人住所地或财产所在地中级法院管辖
  3. 非纽约公约裁决仍可按互惠原则
