# 七 法人

法人住所主张：

  1. 管理中心或主事务所所在地（我国采用）
  2. 实际从事经营活动所在地
  3. 依其章程规定
  4. 成立地

外国法人的认许：指外国法人以法律人格者在内国从事民商事活动的认可。

外国法人认许程序：

  1. 特别，特别登记
  2. 概括，对某一特定外国的法人
  3. 一般
  4. 分别，分门别类

我国外商活动方式：

  1. 临时来华
  2. 直接投资
  3. 设立分公司

法人属人法：指决定法人权利能力和行为能力的准据法。（主要采用法人国籍国法说）
在具体能力上，还受内国的外国人法的约束。

我国对外国法人采注册登记地定其国籍，并以该国法律为其属人法。
和登记地不一致的，可适用主营业地法（经常居所地）

法人属人法的适用范围：

  1. 法人的成立和性质
  2. 法人的权利能力
  3. 法人的内部体制和对外关系
  4. 法人的解散
  5. 法人的合并分立对前法人的债务继承
