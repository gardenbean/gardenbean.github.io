# 四 冲突规范运用中的一般问题

反致种类：

  1. 反致：指甲国（法院国）根据本国冲突规范指引乙国法作准据法时，认为应包括乙国的冲突法，而依乙国冲突规范却应适用甲国的实体法作准据法。（一级反致）
  2. 转致：指依甲国冲突规范 -> 乙国法（包括冲突法）-> 丙国实体法（二级反致）
  3. 间接反致：指依甲国冲突规范 -> 乙国法（包括冲突法）-> 丙国法（包括冲突法）-> 甲国实体法

反致产生原因：

  1. 各国对本国冲突规范指引的外国法的范围理解不同（认为包括冲突法）
  2. 对同一涉外民事关系规定了不同的连结点
  3. 有相互指定的致送关系发生

我国立法排除反致。

先决问题：指在解决争讼问题（本问题或主要问题）时，以首先解决另一个问题（先决问题）为条件。

先决问题构成条件：

  1. 本问题应外国法为准据法
  2. 该问题可单独提出，并有冲突规则可适用
  3. 依本问题准据法所属国和法院地国冲突规则，会选择出不同国家法作准据法，使本问题判决不同

我国采用法院地国冲突规范确定先决问题的准据法。

法律规避：指当事人为了利用某一冲突规范，故意制造连结点，以避开本应适用的准据法，对自己有利的法律得以适用的一种逃法行为。

法律规避构成要件：

  1. 主观上，有目的、故意的
  2. 规避对象上，是本应适用的强行法
  3. 行为方式上，改变或制造连结点
  4. 客观结果上，达到了对自己有利法律的目的

法律规避我国当作无效法律行为。

外国法的查明方法：

  1. 当事人举证证明（我国采用）
  2. 依职权查明，无须当事人举证
  3. 依职权查明，当事人负协助义务

外国法不能查明的法律适用：

  1. 直接适用内国法（大多数，包括中国）
  2. 推定内外法相同，故而适用内国法
  3. 驳回当事人诉讼请求
  4. 适用与本应适用的外国法相似的法律

公共秩序：指依本国冲突规范适用某一外国法作准据法时，因其适用结果与法院国的重大利益、基本政策、基本道德观念或法律基本原则相抵触，从而可拒绝适用的一种保留制度。

美国学者认为公共政策的场合：

  1. 外国法的适用违背文明国家的道德
  2. 违反法院地的禁止性规定和重要政策
  3. 外国法中的禁止性规定未获得法院地的确认

我国公共秩序间接限制（强行性规定）：

  1. 涉及劳动者权益保护
  2. 涉及公共卫生安全
  3. 涉及环境安全
  4. 涉及金融安全（如外汇管制）
  5. 涉及反垄断、反倾销
