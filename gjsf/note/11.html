# 十一 合同之债法律适用

合同：指当事人之间设立、变更或终止债权债务关系的协议。

合同法律适用主张：
同一论：对合同所有问题以及所有合同都适用一个法律。
分割论：对同一合同的不同问题适当分割分别确定其适用法。
客观论：以某种固定的场所因素作为连结点
主观论：以当事人双方意思（主导）

合同准据法：指适用于合同成立和效力的法律。

合同法律适用发展阶段处于以意思自治原则为主、最密切联系原则为辅。

意思自治原则的限制：

  1. 强制性规则的限制
  2. 公共秩序的限制
  3. 合同特殊问题的限制（如缔约能力，合同形式）
  4. 特殊合同的限制（如不动产合同，消费者合同）

特征性履行：指未选择合同准据法时，适用与合同特征性债务履行人最密切联系法。（住所地法）

合同形式：指当事人订立合同的意思表示的方式。
缔约能力：指当事人订立和履行合同的行为能力。
