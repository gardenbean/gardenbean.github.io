# 一 绪论

涉外民事关系：指在民事关系的主体、客体和权利义务据以发生的法律事实等因素中至少有一个外国因素的民事关系。
法律冲突：指同一涉外民事关系因所涉各国立法不同且都有可能对它进行管辖而产生的法律适用冲突。 区际：指一国内部不同地区的法律制度之间的冲突。 人际：指一国之内适用于不同宗教、种族、不同阶级的人的法律之间的冲突。 时际：指可能影响同一涉外民事关系的新旧前后法律之间的冲突。

国际私法的名称：大陆法系多称为国际私法，英美多称为冲突法。

国际私法的定义：以涉外民事关系为调整对象，以解决法律冲突为中心任务，以冲突规范为最基本的规范，同时包括外国人民事法律地位的规范、避免法律冲突的统一实体规范以及国际民事诉讼与仲裁程序规范在内的一个独立的法律部门。

构成国际惯例条件：（另一部分是国际贸易惯例）

  1. 经长期普遍实践而形成为通则，即物质因素
  2. 经国家或当事人接受为法律，即心理因素

可认定涉外民事关系的情形：

  1. 有一方是外国公民、法人、其他组织、无国籍人
  2. 有一方经常居住地在领域外
  3. 标的物在领域外
  4. 法律事实发生在领域外
  5. 其他

法律冲突产生原因：

  1. 现实生活中大量出现涉外因素的民事关系
  2. 各国民法规定不同
  3. 司法权的独立
  4. 国家为了发展对外民商事关系，必须承认内外法平等，亦即有必要在一定范围内承认外国法的域外效力

法律冲突解决的历史发展阶段：

  1. 依本国的冲突规范解决法律冲突
  2. 依统一冲突规范解决法律冲突
  3. 依统一实体法解决法律冲突

国际私法的范围：

  1. 外国人民事法律地位规范（冲突前提）
  2. 冲突规范（最基本）
  3. 统一实体规范
  4. 国际民事诉讼与国际商事仲裁规范

国际私法的渊源：

  1. 国内立法（最主要）（外国人民地规范，冲突规范，国际民商事程序规范）
  2. 国内判例
  3. 国际条约（巴黎公约，伯尔尼公约，马德里协定及其议定书，世界版权公约，知识产权协定）（但知识产权领域的国际条约已经转化或需要转化为国内法的除外。）
  4. 国际惯例
  5. 国际私法之原则和一般法理及学说

国际私法的基本原则：

  1. 主权原则（最基本）
  2. 平等互利
  3. 国际协调与合作
  4. 保护弱方当事人合法权益
